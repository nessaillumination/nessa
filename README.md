## 🛠️ Tech Stack  

This project is built using state-of-the-art technologies:  

- **Backend**: Node.js  
- **Frontend**: React.js  
- **Database**: MongoDB  
- **CI/CD**: GitHub Actions  
- **Monitoring**: Datadog  
- **Logging**: Winston  
- **Linting**: ESLint  
- **Security**: Helmet, Rate Limiting  
- **Caching**: Redis
